# 자바 IO(Inout, Output)
- 자바에서는 입출력을 위한 다양한 클래스를 제공한다.
  - Byte 단위의 입출력 클래스
    - InputStream, OutputStream
  - Char 단위의 입출력 클래스
    - Reader, Writer

- 파일 입출력
  - FileInputStream, FileOutputStream
  - FileReader, FileWriter

- 배열 입출력
  - ByteArrayInputStream, ByteArrayOutputStream
  - CharReader, CharWriter
